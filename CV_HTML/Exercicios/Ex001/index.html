No capítulo 5 - aula 1:
Aprendi que você pode escrever um parágrafo de qualquer jeito, até mesmo pulando linhas, oque realmente importa é se estão dentro das tags (assim que são chamados todos essas palavras ou letrinhas entre o sinal de menor (<) e maior(>)).

Para escrever um titulo que nem o la de cima você usa o <h1>.

Para você escrever parágrafos que nem esses, você pode ultilizar a tag <p>.

Para quebrar alguma parte específica no texto, você pode usar o <b>.
